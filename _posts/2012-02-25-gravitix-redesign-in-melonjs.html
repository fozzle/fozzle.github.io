I've been dabbling in game design recently. I've been working with C#/XNA to make a very basic platformer, just to learn the basics. I quickly realized that building a game engine from the ground up (or in this case, off a framework) was a bigger task than previously anticipated. With school, work, and other projects competing for time, I put it on the shelf.

Then, I stumbled upon the HTML5 game engine, MelonJS, the other day. I was pretty blown away by how easy it was to use. It integrates closely with Tiled, provides basic object entities, and a lot of other game-specific boilerplate. Not to mention, it's HTML5. That's hip, right?

I'm starting anew with the intent to resurrect a game a friend and I worked on back in middle school. When we were kids, we made all kinds of half-finished games using the drag-n-drop program GameMaker. We have the abandoned childhood websites to prove it!

These should be good for a laugh. About the design...well...I plead innocence. I was 12.

Biohazard Game Studios - Our "companies" website. Notice under "games", none have been completed yet. I guess some things never change...until now!

RscarBHG - My mate's personal website. He kept toying around with GameMaker long after most of us stopped. He is actually the true creator of Gravitix.

Note: You'll probably not be able to run any of these games unless you have WindowsXP or virtualize WindowsXP.
